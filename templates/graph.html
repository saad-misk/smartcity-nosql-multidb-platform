📍 Departments Sharing Districts
Find departments that cover the same geographic area
CYPHER
MATCH (d1:Department)-[:COVERS]->(dist:District) <-[:COVERS]-(d2:Department) WHERE d1.id < d2.id RETURN d1.name AS dept_a, d2.name AS dept_b, dist.name AS shared_district ORDER BY dist.name

🏆 Top Technicians by Performance
Ranked by number of resolved service requests
CYPHER
MATCH (t:Technician)-[:RESOLVED]->(r:ServiceRequest) RETURN t.name AS technician, COUNT(r) AS resolved ORDER BY resolved DESC LIMIT 5

👷 Technicians per Department
Staff listing with resolved request counts
CYPHER
MATCH (t:Technician)-[:WORKS_FOR]->(d:Department) OPTIONAL MATCH (t)-[:RESOLVED]->(r:ServiceRequest) RETURN t.name AS technician, d.name AS department, COUNT(r) AS resolved_count ORDER BY resolved_count DESC

🗺️ Cross-District Reporters
Citizens filing reports outside their home district
CYPHER
MATCH (c:Citizen)-[:FILED]->(r:ServiceRequest) -[:LOCATED_IN]->(d:District) WHERE NOT (c)-[:LIVES_IN]->(d) RETURN c.name AS citizen, d.name AS reported_district LIMIT 20
