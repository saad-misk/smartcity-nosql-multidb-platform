🗺️ Department Coverage Map
Which departments cover which districts (COLLECT aggregation)
CYPHER
MATCH (dept:Department)-[:COVERS]->(dist:District) RETURN dept.name, COLLECT(dist.name) ORDER BY dept.name

Ranked by resolved service requests

👤 Citizen Request Journey
Full interaction chain for a citizen
CYPHER
MATCH (c:Citizen)-[:FILED]->(r:ServiceRequest) -[:ASSIGNED_TO]->(dept:Department) OPTIONAL MATCH (t:Technician)-[:RESOLVED]->(r) RETURN c.name, r.category, r.status, dept.name, t.name

📍 Departments Sharing Districts
Departments that cover the same geographic area
CYPHER
MATCH (d1:Department)-[:COVERS]->(dist:District) <-[:COVERS]-(d2:Department) WHERE d1.id < d2.id RETURN d1.name, d2.name, dist.name

🗺️ Cross-District Reporters
Citizens filing reports outside their home district
CYPHER
MATCH (c:Citizen)-[:FILED]->(r:ServiceRequest) -[:LOCATED_IN]->(d:District) WHERE NOT (c)-[:LIVES_IN]->(d) RETURN c.name, d.name

📊 Department Efficiency
Resolved vs total — resolution rate percentage
CYPHER
MATCH (r:ServiceRequest)-[:ASSIGNED_TO]->(dept:Department) WITH dept, COUNT(r) AS total, SUM(CASE WHEN r.status='RESOLVED' THEN 1 ELSE 0 END) AS resolved RETURN dept.name, total, resolved, ROUND(100.0*resolved/total, 1) AS rate_pct

💥 Impact Analysis
If a department shuts down — who is affected?
CYPHER
MATCH (dept:Department)<-[:ASSIGNED_TO]-(r:ServiceRequest) <-[:FILED]-(c:Citizen) MATCH (dept)-[:COVERS]->(d:District) RETURN COLLECT(DISTINCT c.name), COLLECT(DISTINCT d.name), COUNT(DISTINCT r)

🔍 Collaboration Gaps
Departments that SHOULD collaborate but don't yet — negative pattern matching
CYPHER
MATCH (d1:Department)-[:COVERS]->(dist:District) <-[:COVERS]-(d2:Department) WHERE d1.id < d2.id AND NOT (d1)-[:COLLABORATES_WITH]-(d2) RETURN d1.name, d2.name, COLLECT(dist.name), COUNT(dist)

🏗️ District Connectivity Score
How connected is each district — departments, citizens, requests
CYPHER
MATCH (d:District) OPTIONAL MATCH (d)<-[:COVERS]-(dept:Department) OPTIONAL MATCH (d)<-[:LIVES_IN]-(c:Citizen) OPTIONAL MATCH (d)<-[:LOCATED_IN]-(r:ServiceRequest) RETURN d.name, COUNT(DISTINCT dept), COUNT(DISTINCT c), COUNT(DISTINCT r)
